This repository contains examples to build your own Integration Portal. You can also leverage AI tools like Cursor or Heroku Vibes to build and deploy the portal.


**Sample prompt for Heroku Vibes**

```
Create aHome Banking portal application following the instructions here: https://github.com/diegor/integration-portal-api/blob/main/prompts/HerokuSample.md

Use this variables 
SF_URL: <Put your Salesforce URL>
CLIENT_ID: <Put your client Id>
CLIENT_SECRET: <Put your client secret>

PortalId is <Put your portal Id>

Make sure it supports CORS to embed it in heroku vibes site

Make it look similar to https://www.citi.com
```
